# Variants

Variants allows you to create multiple versions of the same component.

## Adding variants

```js /color/
import { sv } from 'stylex-variants';

const buttonVariants = sv({
  base: styles.base,
  variants: {
    color: {
      primary: styles.primary,
      error: styles.error,
      succes: styles.succes,
    },
  },
});

buttonVariants({ color: 'error' });
```

## Multiple variants

```js /color/ /size/
import { sv } from 'stylex-variants';

const buttonVariants = sv({
  base: styles.base,
  variants: {
    color: {
      primary: styles.primary,
      error: styles.error,
      succes: styles.succes,
    },
    size: {
      sm: styles.sm,
      md: styles.md,
      lg: styles.lg,
    },
  },
});

buttonVariants({ color: 'error', size: 'lg' });
```

## Default variants

```js /color/ /size/
import { sv } from 'stylex-variants';

const buttonVariants = sv({
  base: styles.base,
  variants: {
    color: {
      primary: styles.primary,
      error: styles.error,
      succes: styles.succes,
    },
    size: {
      sm: styles.sm,
      md: styles.md,
      lg: styles.lg,
    },
  },
  defaultVariants: {
    color: 'primary',
    size: 'md',
  },
});

buttonVariants();
```

## Boolean variants

```js /color/ /size/
import { sv, type VariantProps } from 'stylex-variants';

const buttonVariants = sv({
  base: styles.base,
  variants: {
    disabled: {
      true: styles.inactive,
      false: styles.active,
    },
  },
});

buttonVariants();
```

<Callout type="info">
  The default variant for a boolean variant is false by default.
</Callout>

## Required variants

```js /required: true,/
import { sv } from 'stylex-variants';

const buttonVariants = sv({
  base: styles.base,
  variants: {
    color: {
      primary: styles.primary,
      error: styles.error,
      succes: styles.succes,
    },
    size: {
      sm: styles.sm,
      md: styles.md,
      lg: styles.lg,
      required: true,
    },
  },
});

type ButtonVariants = VariantProps<typeof buttonVariants>;

/*
type ButtonVariants = {
  size: 'sm' | 'md' | 'lg',
  color?: 'primary' | 'error' | 'succes' | undefined,
};
*/
```
