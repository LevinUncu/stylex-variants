# Installation

**StyleX Variants** requires you to have `StyleX` installed.<br />
You can follow the [StyleX Installation Guide](https://stylexjs.com/docs/learn/installation/).

To use **StyleX Variants** in your project, you can install it as a dependency:

<Tabs items={['npm', 'yarn', 'pnpm']}>
  <Tab>

    ```bash copy
    npm add stylex-variants
    ```

  </Tab>
  <Tab>

    ```bash copy
    yarn add stylex-variants
    ```

  </Tab>
  <Tab>

    ```bash copy
    pnpm add stylex-variants
    ```

  </Tab>
</Tabs>
